ProductManagement este un API .NET 9 conceput pentru a asigura o validare riguroasa a datelor in procesul de creare a produselor. Sistemul implementeaza reguli complexe de business (verificari asincrone de unicitate, constrangeri de pret si stoc in functie de categorie) și ofera o vizibilitate completă prin logare structurata.

API-ul expune operațiile esentiale de management: crearea produselor (POST), listarea tuturor produselor (GET) si obtinerea detaliilor despre un anumit produs (GET by ID).
